What is the Weather like as we approach the equator?

"It gets hotter..." But, How would we prove it? we will be using a Python script to visualize the weather of 500+ cities across the world of varying distance from the equator. To accomplish this, we'll be utilizing a simple Python library and the OpenWeatherMap API, to create a representative model of weather across world cities. The objective is to build a series of scatter plots to showcase the correlations between: Temperature (F) vs. Latitude, Humidity (%) vs. Latitude, Cloudiness (%) vs. Latitude, Wind Speed (mph) vs. Latitude.

Visualizations

The Data have been manipulated with (https://pypi.python.org/pypi/citipy), using Jupyter Notebook, MatplotLib and Open Weather Map API (https://openweathermap.org/api)
